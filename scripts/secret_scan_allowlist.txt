# secret_scan.py allowlist
# One entry per line. Lines starting with '#' are comments.
#
#   <regex>        -> any secret-shaped match whose text matches this regex is
#                     downgraded to ALLOWED (not reported).
#   path:<glob>    -> files matching this glob are skipped entirely.
#
# Keep this list SHORT and REVIEWED. Every entry is a deliberate decision that a
# given pattern is a false positive or an intentionally-public value. Do not add
# a real secret here -- gitignore the file or remove the secret instead.
#
# Examples (commented out):
#   8\.8\.8\.8                 # Google public DNS referenced in a networking doc
#   path:docs/examples/*.md    # docs that intentionally show sample tokens
